# 🚀 Lazarev Clone Website

A modern and animated frontend website inspired by the Lazarev digital product design agency website.  
This project is built using HTML, CSS, JavaScript, GSAP, and Locomotive Scroll with smooth scrolling and interactive animations.

## ✨ Features

- Smooth scrolling using Locomotive Scroll
- Advanced GSAP animations
- Interactive navigation menu
- Video play animations
- Hover effects on sections
- Responsive modern UI
- ScrollTrigger animations
- Animated loading screen

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- GSAP
- ScrollTrigger
- Locomotive Scroll
- Remix Icons

## ▶️ How to Run the Project

1. Download or clone the repository

```bash
git clone https://github.com/your-username/lazarev-clone.git
```

2. Open the project folder

3. Run the `index.html` file in your browser
